## 🎯 核心功能

### 🫧 呼吸灯效果
- **正弦波算法**：平滑的明暗变化
- **可配置频率**：1-2 秒/周期（默认 1.0 Hz）
- **可配置强度**：0-1 范围（默认 0.7）
- **50ms 刷新率**：流畅动画体验

### 🎨 智能状态切换
- 🟢 **正常状态**：绿色呼吸（#00ff88）
- 🟡 **警告状态**：橙色呼吸（#ffaa00）- 最后 10 秒
- 🔴 **完成状态**：红色加速呼吸（#ff3333）- 时间到，频率 x3

### ⚙️ 配置系统
```json
{
  "breath_light": {
    "enabled": true,
    "mode": "digital",
    "frequency": 1.0,
    "intensity": 0.7,
    "color_scheme": {
      "normal": "#00ff88",
      "warning": "#ffaa00",
      "completed": "#ff3333"
    },
    "accelerate_on_complete": true
  }
}
```
