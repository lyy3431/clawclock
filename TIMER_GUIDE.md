# ClawClock 倒计时功能使用指南

## 📖 功能概述

ClawClock v1.4.0 新增了完整的倒计时功能，支持：
- ⏱️ 精确倒计时（毫秒级）
- 🎯 预设时间快捷设置（番茄钟、休息时间等）
- 🔊 声音提醒
- 💡 视觉提醒（闪烁效果）
- 💾 配置持久化
- ⏸️ 暂停/继续功能
- 🔄 一键重置

## 🚀 快速开始

### 1. 启动应用

```bash
cd /home/lenovo/AItest/clawclock
python3 clock.py
```

### 2. 切换到倒计时模式

在主界面中，选择 **"⏳ 倒计时"** 单选按钮。

### 3. 设置倒计时

#### 方法一：使用预设时间（推荐）

点击预设时间按钮快速设置：

| 预设名称 | 时长 | 使用场景 |
|----------|------|----------|
| 🍅 **番茄钟** | 25 分钟 | 专注工作、深度学习 |
| ☕ **短休息** | 5 分钟 | 番茄钟间隔休息 |
| 🛌 **长休息** | 15 分钟 | 长时间工作后恢复 |
| ⏱️ **5 分钟** | 5 分钟 | 快速任务、热身 |
| ⏱️ **10 分钟** | 10 分钟 | 中等任务、冥想 |
| ⏱️ **30 分钟** | 30 分钟 | 会议、练习、复习 |

#### 方法二：自定义时间

手动输入任意时长：
1. 在时间输入框中直接输入小时、分钟、秒
2. 或使用上下箭头调整数值
3. 点击"▶️ 开始"启动倒计时

**开发者扩展预设**：
在 timer.py 的 `PRESET_TIMES` 列表中添加自定义预设：

```python
PRESET_TIMES = [
    PresetTime("番茄钟", minutes=25),
    PresetTime("短休息", minutes=5),
    PresetTime("长休息", minutes=15),
    PresetTime("自定义 45 分钟", minutes=45),  # 添加自定义
    PresetTime("会议时间", minutes=60),
    PresetTime("冥想练习", minutes=10),
]
```

### 4. 控制倒计时

- **▶️ 开始** - 开始倒计时
- **⏸️ 暂停** - 暂停倒计时（可继续）
- **🔄 重置** - 重置为初始设置时间
- **🔊 声音** - 开启/关闭声音提醒

### 5. 时间到提醒

当倒计时结束时：
1. **视觉提醒**：时间显示区域会红色闪烁
2. **声音提醒**：播放系统提示音（可关闭）
3. **状态提示**：显示"⏰ 时间到！"

**关闭提醒**：点击"🔊 声音"按钮或关闭应用

## 🔧 高级用法

### 在代码中使用 Timer 类

```python
from timer import Timer, PresetTime

# 创建倒计时
timer = Timer(hours=0, minutes=25, seconds=0)

# 或使用预设
timer.set_preset(PresetTime("番茄钟", minutes=25))

# 设置完成回调
def on_complete():
    print("时间到！")

timer.on_complete = on_complete

# 开始倒计时
timer.start()

# 获取剩余时间
remaining = timer.get_remaining()
print(f"剩余：{timer.format_time(remaining)}")

# 暂停
timer.stop()

# 继续
timer.start()

# 重置
timer.reset()
```

### 使用 TimerWidget 创建独立窗口

```python
from timer import create_timer_window

# 创建独立的倒计时窗口
root = create_timer_window()
root.mainloop()
```

## 🎯 选择建议

**使用秒表**：
- 需要记录总耗时（如跑步、游泳）
- 需要计圈功能（如间歇训练）
- 不确定具体时长，想看看用了多久

**使用倒计时**：
- 有明确的目标时长（如番茄钟 25 分钟）
- 需要时间到提醒（如烹饪、会议）
- 想控制任务时长，避免超时
- 使用预设时间快速设置

## 🐛 常见问题

### Q: 倒计时不准确？
A: 倒计时使用系统时间计算，确保系统时间准确。

### Q: 没有声音提醒？
A: 检查：
1. 声音开关是否开启
2. 系统音量是否正常
3. 某些系统可能需要安装额外的音频库

### Q: 配置没有保存？
A: 检查 `config.json` 文件权限，确保应用有写入权限。
